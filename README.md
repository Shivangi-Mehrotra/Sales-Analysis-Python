# Sales-Analysis-Python
📌 Project Overview

Conducted exploratory data analysis on a sales dataset using Python to identify trends, monthly performance, and key business insights through data visualization.

⸻

🛠 Tools & Technologies
	•	Python
	•	Pandas
	•	NumPy
	•	Matplotlib
	•	Seaborn
	•	Scikit-learn (MinMaxScaler)

⸻

📈 Key Analysis Performed
	•	Data cleaning and preprocessing of sales dataset
	•	Normalization of sales and unit data using MinMaxScaler
	•	Time-series analysis of daily sales and units
	•	Monthly sales comparison (October, November, December)
	•	Visualization of sales trends and distributions

⸻

📊 Key Insights
	•	Total Sales: 340M+
	•	Total Units Sold: 136K+
	•	Highest Sales Day: 14 Dec 2020
	•	December recorded the highest average sales compared to other months.
